# Full-Stack Project

Expense Tracker website where you can input categorised expenses and see a chart visualization of the distribution.

## How to run

To run locally, first install backend and frontend dependencies with npm install in their corresponding directories. Then you need to create a **.env** file that looks something like this:
NODE_ENV=development
PORT=8000
MONGO_URI=your mongodb connection string
JWT_SECRET= your jwt secret key
